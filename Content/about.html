This application was built for the Spring Fling silent auction fundraiser of the Philly Free School [http://www.phillyfreeschool.org/] in Philadelphia, PA.

It is based on the following open source project:


BidHub-Android [https://github.com/HubSpot/BidHub-Android] by Joshua Tsuji / HubSpot

The main functionality of the Spring Fling app is taken from the BidHub Android app. However, that app used the Parse MBaaS backend, which was no longer accepting new accounts when I began this project. Parse will be fully retired on January 28, 2017.
